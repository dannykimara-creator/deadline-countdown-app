# deadline-countdown-app
A Python command-line application that calculates the remaining time until a specified deadline.

## Features

- Accepts a custom deadline from the user

- Calculates remaining days, hours and minutes

- Displays countdown in real time

- Handles invalid date inputs gracefully

## Technologies

- Python 3

- datetime module

- time module

## Skills Demonstrated

- Working with dates and time

- User input validation

- Loops

- Functions

- Error handling

## Future Improvements

- GUI version using Tkinter

- Desktop notifications

- Multiple saved deadlines
